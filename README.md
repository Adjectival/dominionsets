# Dominion Sets
App to help manage all the ways to play the card game Dominion.

## Development recipe
- clone this repo
- $ cd this_repo
- $ python3 -m venv working
- $ source working/bin/activate
- (working)$ pip install -r requirements.txt
- (working)$ python manage.py runserver
